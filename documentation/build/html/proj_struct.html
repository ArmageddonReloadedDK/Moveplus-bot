Project structure¶

This project consists of different modules.

List of them:

  1. bot.py - main body of the bot

  2. dbworker.py - file with functions for getting/changing Vedis database values

  3. config.py - file with states values and questions

  4. functions.py - functions wich makes queries to Posgresql database
