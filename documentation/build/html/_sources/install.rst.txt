Install
============

First of all you need to download all files from `bot repository`_ and  `web repository`_.

Then you need to install `python=3.6`_ and some modules.

.. _python=3.6: https://realpython.com/installing-python/
.. _web repository: https://github.com/noDGodyaev/django_for_events
.. _bot repository: https://github.com/ArmageddonReloadedDK/Telegram-event-registration-bot

Do in command line::

 pip install telebot 
 
 pip install cython vedis # DBMS for file-database
 
 pip install psycopg2 # tool to work with postgresql 
 
 pip install Django  # main web framework
 
 pip install Model-admin-reoder # for features in django-admin app

 pip install Rest Framework # for serialisation funcionality (for ajax in html forms)
 
Install any relational DBMS. In our project we use PostgreSQL.
